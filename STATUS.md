## What This Repo Is

AI-Trainer-OBLIVION is a **deliberate teaser stub** — not an unfinished project.

The README and landing page (`docs/index.html`) are intentionally scoped: "No release date yet." / "Shipped when it's ready, not before." This is a scarcity-and-signal play: stake the name, stake the arc, invite the right audience to watch. The curriculum has not been written yet. That is by design.

Comparing to [AI-Trainer-MAX](https://github.com/thebardchat/AI-Trainer-MAX): MAX shipped with 36 modules across 5 phases (144 files). OBLIVION's 10 planned topics are a rough outline — not modules. They require original research, hardware testing, and hands-on validation before any content is written.
